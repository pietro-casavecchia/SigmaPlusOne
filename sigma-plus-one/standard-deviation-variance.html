Standard Deviation & Variance

  • Wikipedia: Standard Deviation
  • Wikipedia: Variance

The Standard Deviation is a measure of the amount of variation or dispersion of a set of values. A low standard deviation indicates that the values tend to be close to the mean. There are two types of Standard Deviation. The Population Standard Deviation is a parameter, which is a fixed value calculated from every individual in the population. The Sample Standard Deviation is a statistic. This means that it is calculated from only some of the individuals in a population. The advantage of variance is that it treats all deviations from the mean as the same regardless of their direction. Variance has not anymore the unit of measure of the data set thus is more complicate to visualize and apply in a real-world sense.

$$ \sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N} (x_{i} - \mu)^2} $$
  • \(\sigma\) = Population Standard Deviation
  • \(x_{i}\) = The value with index i form the data set
  • \(\mu\) = The mean or expected value
  • \(N\) = Number of values in data set
$$ s = \sqrt{\frac{1}{N - 1}\sum_{i=1}^{N} (x_{i} - \overline{x})^2} $$
  • \(s\) = Sample Standard Deviation
  • \(x_{i}\) = A value with index i form the sample
  • \(\overline{x}\) = The mean of the sample
  • \(N\) = Sample size
