Summation & Product

  • Wikipedia: Summation
  • Wikipedia: Product

Summation with the capital Greek letter Sigma, is the sum of all the elements of the set. Product intended as the product with the capital Greek letter Pi, is the product of all the elements of the set.

$$ \Sigma = \sum_{i=1}^{n} x_{i} = x_{1} + x_{2} + \ldots + x_{n} $$
  • \(\Sigma\) = Sum
  • \(x_{i}\) = The value with index i form the data set
  • \(n\) = Number of values in data set
$$ \Pi = \prod_{i=1}^{n} x_{i} = x_{1} \cdot x_{2} \cdot \ldots \cdot x_{n} $$
  • \(\Pi\) = Product
  • \(x_{i}\) = The value with index i form the data set
  • \(n\) = Number of values in data set
